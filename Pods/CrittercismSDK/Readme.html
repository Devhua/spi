What's in this Download?

This zip archive should contain the following files:

  • Readme.html (this file)

  • CrittercismSDK/

    The headers and static library file you will need to link Crittercism in to your application.

Documentation

Release Notes

A list of all SDK changes can be found online at http://docs.crittercism.com/release_notes/release_notes_ios.html.

Support
